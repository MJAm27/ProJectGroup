# 🍰 BakeryLand

ระบบสั่งซื้อสินค้าเบเกอรีผ่านเว็บไซต์ ซึ่งพัฒนาขึ้นเป็นส่วนหนึ่งของรายวิชา **Web Development** โดยผู้ใช้งานสามารถเลือกซื้อสินค้า จัดการข้อมูลส่วนตัว และสั่งซื้อสินค้าได้ผ่านระบบ

### 🛠 เทคโนโลยีที่ใช้

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* Bootstrap

### 👨‍💻 หน้าที่ที่รับผิดชอบ

* ออกแบบและพัฒนาหน้า **โปรไฟล์ผู้ใช้งาน (User Profile)** ฝั่ง Front-end
* พัฒนา Layout แบบ Responsive สำหรับหน้าจัดการข้อมูลส่วนตัว
* ร่วมออกแบบโครงสร้างฐานข้อมูลและความสัมพันธ์ของตารางใน MySQL
* ทำงานร่วมกับสมาชิกในทีมเพื่อเชื่อมต่อระบบและทดสอบการทำงาน

### ✨ ฟังก์ชันหลัก

* ระบบสมัครสมาชิกและเข้าสู่ระบบ
* แสดงรายการสินค้า
* ระบบตะกร้าสินค้า
* จัดการข้อมูลผู้ใช้งาน
* ระบบสั่งซื้อสินค้า

# 🗄️ Database Project

โปรเจกต์ด้านการออกแบบฐานข้อมูลเชิงสัมพันธ์ (Relational Database) โดยมุ่งเน้นการสร้างโครงสร้างฐานข้อมูลที่มีประสิทธิภาพ มีการทำ Normalization และออกแบบความสัมพันธ์ของข้อมูลอย่างเหมาะสม

### 🛠 เทคโนโลยีที่ใช้

* MySQL
* SQL

### 👨‍💻 หน้าที่ที่รับผิดชอบ

* ออกแบบโครงสร้างฐานข้อมูลทั้งหมดของระบบ
* จัดทำ ER Diagram และออกแบบ Schema ของฐานข้อมูล
* กำหนด Primary Key, Foreign Key และความสัมพันธ์ระหว่างตาราง
* ประยุกต์ใช้หลักการ Database Normalization เพื่อลดความซ้ำซ้อนของข้อมูล
* พัฒนา SQL Script สำหรับสร้างฐานข้อมูล

### ✨ ฟังก์ชันหลัก

* การออกแบบฐานข้อมูลแบบ Normalization
* โครงสร้างฐานข้อมูลเชิงสัมพันธ์
* SQL Schema
* การจัดเก็บข้อมูลอย่างมีประสิทธิภาพ

# 🎁 Gift Basket

โปรเจกต์ต้นแบบ (Prototype) สำหรับระบบสั่งซื้อกระเช้าของขวัญ โดยเน้นการวางแผนและออกแบบระบบก่อนเริ่มพัฒนา ทั้งในด้านประสบการณ์ผู้ใช้งานและโครงสร้างฐานข้อมูล

### 🛠 เทคโนโลยีที่ใช้

* Figma
* MySQL

### 👨‍💻 หน้าที่ที่รับผิดชอบ

* ออกแบบ Prototype ของระบบ (UI/UX)
* จัดทำ User Flow และ Wireframe
* ออกแบบโครงสร้างฐานข้อมูลและความสัมพันธ์ของข้อมูล
* วางแผนโครงสร้างระบบร่วมกับสมาชิกในทีม

### ✨ ฟังก์ชันหลัก

* ต้นแบบ UI/UX
* User Flow
* การออกแบบฐานข้อมูล
* การวางแผนโครงสร้างระบบ

# 🎮 Project Game

โปรเจกต์เพื่อการเรียนรู้ โดยศึกษาจากตัวอย่างบน YouTube และนำมาพัฒนาต่อยอด เพิ่มฟังก์ชันและปรับปรุงการทำงานให้แตกต่างจากต้นแบบ

### 🛠 เทคโนโลยีที่ใช้

* HTML
* CSS
* JavaScript

### 👨‍💻 หน้าที่ที่รับผิดชอบ

* ศึกษาและพัฒนาโปรเจกต์จากตัวอย่าง
* เพิ่มฟังก์ชันการเล่นใหม่จากต้นแบบ
* ปรับปรุงตรรกะการทำงานของเกม
* ปรับแต่งหน้าตาและประสบการณ์การใช้งาน

### ✨ ฟังก์ชันหลัก

* ระบบเกมแบบโต้ตอบ
* ปรับปรุงกลไกการเล่น
* พัฒนาหน้าตาอินเทอร์เฟซ
* เพิ่มฟังก์ชันการทำงานใหม่
